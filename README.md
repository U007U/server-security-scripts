# README.md – Как настроить безопасность сервера (кратко)

1. Скачать скрипт:
   ```bash
   wget https://raw.githubusercontent.com/U007U/server-security-scripts/main/secure_server.sh
Сделать исполняемым:

bash
chmod +x secure_server.sh
Запустить с правами root:

bash
sudo ./secure_server.sh
Выбрать язык: нажми 1 (English) или 2 (Русский).

Ввести имя нового пользователя (например, anya).

Выбрать порт SSH: просто нажми Enter (останется 2222).

Ответить про SSH-ключ:

Если есть ключ → yes, затем вставь публичный ключ (одна строка).

Если нет ключа → no (пароли не отключатся, сможешь добавить ключ позже).

Ввести пароль для нового пользователя (придумай сложный).

После завершения скрипта подключайся к серверу так:

bash
ssh имя_пользователя@89.107.10.79 -p 2222
Готово. Сервер защищён.
