# 🚀 Настройка безопасности сервера (кратко)

Автоматическая защита Ubuntu: SSH-порт 2222, новый пользователь, ключи, firewall.

## 📥 Шаги установки

1. **Создай файл скрипта:**
   ```bash
   nano secure_server.sh
Скопируй содержимое secure_server.sh (выше) и вставь. Сохрани (Ctrl+O, Enter, Ctrl+X).

2. Сделай исполняемым:

bash
chmod +x secure_server.sh
3. Запусти от root:

bash
sudo ./secure_server.sh
4. Следуй подсказкам:

Выбери язык (1/2)

Имя пользователя (например, anya)

Порт SSH (Enter для 2222)

Есть ли SSH-ключ? Если да — вставь публичный ключ (одна строка)

5. Подключись после настройки:

bash
ssh anya@89.107.10.79 -p 2222
(замени anya и IP на свои)

✅ Результат
Root отключён

Сервер защищён (фаервол + fail2ban)

Вход по ключу (если добавил) или паролю
