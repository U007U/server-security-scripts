# Быстрая настройка безопасности сервера

Этот скрипт автоматизирует основные шаги по защите сервера Ubuntu. Он включает:
- Отключение прямого входа root через SSH.
- Изменение порта SSH.
- Настройку Firewall (UFW).
- Создание нового пользователя с правами администратора.
- Активацию автоматических обновлений.
- Настройку Fail2Ban для предотвращения брутфорс-атак.

## Особенности:
- Скрипт выводит информационные сообщения на каждом этапе.
- Поддерживается выбор языка (русский или английский).
- После завершения работы скрипт автоматически удаляет себя.

## Использование:

1. Скачать скрипт:
   ```bash
   wget https://raw.githubusercontent.com/U007U/server-security-scripts/main/secure_server.sh
   ```

2. Сделать файл исполняемым:
   ```bash
   chmod +x secure_server.sh
   ```

3. Запустить скрипт с правами суперпользователя:
   ```bash
   sudo ./secure_server.sh
   ```

4. Следуйте подсказкам для задания пароля нового пользователя.

   **ВАЖНО**: При создании нового пользователя вам будет предложено ввести пароль дважды. Оставьте остальные поля пустыми, если не хотите указывать дополнительную информацию.

5. После завершения работы скрипта, подключитесь к серверу с новым пользователем и новым портом SSH.

6. Проверьте состояние Firewall:
   ```bash
   sudo ufw status
   ```

   Эта команда покажет, какие порты открыты и закрыты. Например, вы должны увидеть что-то подобное:

   ```
   Status: active

   To                         Action      From
   --                         ------      ----
   2222                       ALLOW       Anywhere
   2222 (v6)                  ALLOW       Anywhere (v6)
   ```

   Это означает, что порт 2222 открыт для входящих соединений, а все остальные порты закрыты.
