# 🚀 Настройка безопасности сервера — запуск в 3 команды

## 1. Создать файл скрипта

Скопируй и выполни эту команду (вставь целиком, включая `EOF`):

```bash
cat > secure_server.sh << 'EOF'
(СЮДА ВСТАВЬ ВСЁ СОДЕРЖИМОЕ ФАЙЛА secure_server.sh ИЗ ВЫШЕПРИВЕДЁННОГО БЛОКА)
EOF
2. Запустить скрипт
bash
chmod +x secure_server.sh && sudo ./secure_server.sh
3. Ответить на вопросы
Язык (1/2)

Имя нового пользователя (например, anya)

Порт SSH (Enter = 2222)

Есть ли SSH-ключ? Если да — вставь публичный ключ (одна строка)

4. Подключиться к серверу
bash
ssh ИМЯ_ПОЛЬЗОВАТЕЛЯ@89.107.10.79 -p 2222
✅ Результат: Root отключён, фаервол пропускает только SSH, Fail2Ban защищает от брутфорса.

text

---

**Как пользоваться:**  
1. Скопируй содержимое **Файла 2** (`README.md`) — это инструкция.  
2. В инструкции выполни первую команду (`cat > secure_server.sh ...`), но **перед этим замени** строку `(СЮДА ВСТАВЬ...)` на **всё содержимое Файла 1** (`secure_server.sh`).  
   - Удобнее: открой два окна. В одном — текст `secure_server.sh`, в другом — терминал. Скопируй команду `cat > ...` до конца с заменой.

Альтернатива (если лень менять): скопируй `secure_server.sh` вручную через `nano secure_server.sh`, но это уже не «одна команда».

Если хочешь полностью автоматический процесс без ручной правки — скачай скрипт с реального URL (например, загрузи его в github и используй `wget`). В текущем виде это максимально близко к «скопировал и вставил».
