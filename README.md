# Быстрая настройка безопасности сервера

Этот скрипт автоматизирует основные шаги по защите сервера Ubuntu. Он включает:
- Отключение прямого входа root через SSH.
- Изменение порта SSH.
- Настройку Firewall (UFW).
- Создание нового пользователя с правами администратора.
- Активацию автоматических обновлений.

## Особенности:
- Скрипт выводит информационные сообщения на каждом этапе.
- После завершения работы скрипт автоматически удаляет себя.

## Использование:

1. Скачать скрипт:
   ```bash
   wget https://raw.githubusercontent.com/U007U/server-security-scripts/main/secure_server.sh
   ```

2. Сделать файл исполняемым:
   ```bash
   chmod +x secure_server.sh
   ```

3. Запустить скрипт с правами суперпользователя:
   ```bash
   sudo ./secure_server.sh
   ```

   **ВАЖНО**: Для скачивания и запуска скрипта требуются права суперпользователя. Если вы столкнулись с ошибкой "Permission denied", выполните команду `sudo su -` перед скачиванием файла.

4. Следуйте подсказкам для задания пароля нового пользователя.

## Важные замечания:
- Скрипт предполагает использование Ubuntu. Для других дистрибутивов могут потребоваться модификации.
- Рекомендуется ознакомиться с содержанием скрипта перед его применением.
